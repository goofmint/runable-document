Onsen UIの使い方を学びましょう

まずはOnsen UIを使うのに必要なCSS、JavaScriptのタグをHTMLに追加します。実行ボタンを押してください（HTMLの内容がないので、真っ白いウィンドウだけが表示されます）。

      
        <link rel="stylesheet" href="https://unpkg.com/onsenui/css/onsenui.css">
        <link rel="stylesheet" href="https://unpkg.com/onsenui/css/onsen-css-components.min.css">
        <script src="https://unpkg.com/onsenui/js/onsenui.min.js"></script>

では次に試しにボタンを配置してみましょう。 <ons-button> というタグを使います。このタグを使うだけで、スマートフォンっぽいボタンが配置できます。実行ボタンを押してみましょう。

      
        <ons-button onclick="alert('Hello World!')">Click Me</ons-button>

ボタンが表示されましたか？表示されたら、Click Meを押してみましょう。Hello World! というアラートが確認できるはずです。

ナビゲーションを体験する

では次にアプリっぽいUIを実現するのに欠かせないナビゲーションを使ってみましょう。これは <ons-navigator /> タグを使います。

以下のコードを実行してみましょう。まだナビゲーション用のJavaScriptコードは記述していないので動きませんので注意してください。

      
        <ons-navigator swipeable id="myNavigator" page="page1.html"></ons-navigator>

<template id="page1.html">
  <ons-page id="page1">
    <ons-toolbar>
      <div class="center">Page 1</div>
    </ons-toolbar>

    <p>This is the first page.</p>

    <ons-button id="push-button">Push page</ons-button>
  </ons-page>
</template>

<template id="page2.html">
  <ons-page id="page2">
    <ons-toolbar>
      <div class="left"><ons-back-button>Page 1</ons-back-button></div>
      <div class="center"></div>
    </ons-toolbar>

    <p>This is the second page.</p>
  </ons-page>
</template>

では最後にナビゲーション用のJavaScriptを紹介します。以下のようなコードで動きます。早速実行ボタンを押してみましょう。

      
        document.addEventListener('init', function(event) {
          var page = event.target;
          if (page.id === 'page1') {
            page.querySelector('#push-button').onclick = function() {
              document.querySelector('#myNavigator').pushPage('page2.html', {data: {title: 'Page 2'}});
            };
          } else if (page.id === 'page2') {
            page.querySelector('ons-toolbar .center').innerHTML = page.data.title;
          }
        });

表示されたら、Push pageボタンを押してみましょう。ちゃんと画面遷移するのが分かりましたか？
